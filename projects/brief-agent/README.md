# Brief Agent

A document summarization tool for students who need to extract key points from long readings without reading every page.

## MVP (v0.1.0)

- Paste text or upload PDF, DOCX, or PPT
- Bullet-point summary focused on exam prep (optional mini survey)
- Max ~100 pages; Turkish + English
- No persistent storage after processing

## Deferred (v0.2)

Citations, Q&A over document, PDF export, read-aloud
