# Day 3 — Checking Status & History

## What I did
1. Edited `README.md` and checked `git status` (modified, not staged)
2. Staged and observed status change
3. Used `git log` and `git log --oneline --graph`
4. Used `git diff` and `git diff --staged`

## Commands
```powershell
git status
git log --oneline
git log --oneline --graph --all
git diff
git diff --staged
```

## Key idea
`git status` shows working directory vs staging area. `git log` shows history.
