# Day 8 — Connecting to a Remote

## What I did
1. Created remote repository on GitHub
2. Connected local repo with `git remote add origin`
3. Verified with `git remote -v`

## Commands
```powershell
git remote add origin https://github.com/ayse-solmaz/agentic-ai-developer.git
git remote -v
git remote show origin
```

## Key idea
`origin` is the default name for your main remote. Day 8 = connect only; Day 9 = push.

## Lesson learned
Keep practice work in the correct repo — avoid mixing with unrelated project history.
