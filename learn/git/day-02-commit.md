# Day 2 — Staging & Committing

## What I did
1. Created `README.md`
2. Staged with `git add`
3. Committed with `git commit`
4. Wrote a clear commit message

## Commands
```powershell
git add README.md
git status
git commit -m "Add README with project title"
git log --oneline
```

## Key idea
Git saves in two steps: **stage** (`git add`) then **commit** (`git commit`).
