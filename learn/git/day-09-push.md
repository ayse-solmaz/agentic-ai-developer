# Day 9 — Pushing Changes

## What I did
1. Verified remote connection with `git remote -v`
2. Pushed local commits to GitHub with `git push -u origin main`
3. Set upstream tracking for future pushes

## Commands
```powershell
git branch
git remote -v
git push -u origin main
```

## Key idea
`git push` uploads local commits to the remote. `-u` sets upstream so later you can just `git push`.

## Lesson learned
Use one repo for homework progress (`agentic-ai-developer`). Keep branch naming consistent (`main`).
